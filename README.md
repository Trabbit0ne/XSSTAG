# Installation 

``` 
curl -sSL https://raw.githubusercontent.com/Trabbit0ne/XSSTAG/main/main.sh -o $(dirname $(command -v bash))/xsstag && chmod +x $(dirname $(command -v bash))/xsstag
```

Then 

```
xsstag 
```
